# Date Adder

## Project Description

This project automates the process of renaming files by adding today's date (YYYY-MM-DD) to the beginning of each filename inside a folder.

## Purpose

The goal of this project is to practice simple automation using Python and terminal commands.

## How it works

1. A folder contains multiple markdown files.
2. The Python script reads all filenames in the folder.
3. The script adds today's date to the beginning of each filename.
4. The files are renamed automatically.

## Example

Before running the script:

file1.md
file2.md
file3.md

After running the script:

2026-03-08_file1.md
2026-03-08_file2.md
2026-03-08_file3.md
